## What is IPTV?

IPTV (Internet Protocol Television) delivers live TV via the internet instead of cable or satellite.

### Benefits
- Lower cost  
- Global content access  
- Multi-device streaming  
- HD/4K quality  

---

## IPTV Requirements

- Minimum 15 Mbps internet  
- Streaming device (Firestick, Android TV)  
- IPTV Player (IPTV Smarters, XCIPTV)  
- VPN (recommended)  

---

## Installation Guide

### Firestick
1. Enable Unknown Sources  
2. Install Downloader  
3. Install IPTV app  
4. Login with M3U or Xtream Codes  

### Android
1. Open Play Store  
2. Install IPTV app  
3. Add subscription  

### Smart TV
1. Install Smart IPTV / SS IPTV  
2. Upload M3U via MAC address  
3. Restart app
